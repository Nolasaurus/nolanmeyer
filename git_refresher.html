4 areas where git lives local dir staging area local repository remote repository git: sw (included w/linux, macOS) "memory card" for code - save progress as you go checkpointing every so often git init git add files you want to save changes git add . push everything you changed git clone: remote repo --> local repo git checkout: local repo --> local dir git add: local dir --> staging area git commit: staging area --> local repo git push: local repo --> remote repo git pull: remote repo --> local dir git fetch: remote repo --> local dir (get latest updates) git merge: local repo --> local dir (integrate updates) git switch: switch between branches
